    A: Jika komputer Anda 32 bit, 64 bit mungkin tidak berfungsi.

    J: magsty.exe salah didiagnosis sebagai virus karena sifat bat to exe. Silakan lihat saus di atas

  3. Apa itu tombol cepat?

    J: Untuk kenyamanan pengguna, tombol sederhana dibuat untuk dioperasikan dengan memasukkan hanya huruf pertama dari info, start, kill dan full, dan tekan enter.

    kill: bunuh file explorer

    penuh: layar penuh

    shutdown: Matikan komputer

    mulai: Mulai ulang File Explorer

    info: Melihat informasi
